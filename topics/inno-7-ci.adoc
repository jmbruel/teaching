== Usage

A Continuous Integration server allows:

  git [detail]#(pull, checkout, push)#
  *build*
  Archives
  Deploy code
  Execute test suites [detail]#(Junit, Audit, test IHM, ...)#
  Notify results [detail]#(mail, RSS)#
  etc.
